# webJuoppo

Webserverillä toteutettu käyttöliittymäesimerkki JUOPPO/Juomaattille.

Käyttö:
- Kytke JUOPPO Juomaattiin
- Kytke JUOPPO USB-piuhalla tietokoneeseen
- Asenna python 3.x
- Aja komentoriviltä:
```sh
pip install flask
py JUOPPO.py
```
