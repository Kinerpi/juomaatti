# Juoppo

Arduino UNO:lla toimiva yksinkertainen serial ohjain Juomaatille.

```
                            (riviliitin)    
pin 2 = ensimmäinen bitti   (110)
pin 3 = toinen bitti        (111)
pin 4 = kolmas bitti        (113)
pin 5 = EXECUTE             (112)
pin 6 = BUSY                (115)
```
